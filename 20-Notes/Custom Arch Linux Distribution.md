# Custom Arch Linux Distribution

This is a sub-project of the [[Cognitive Internet Project]]. The goal is to create a custom Arch Linux ISO with pre-configurations and setups for the main development laptop.

## Requirements

- Optimized for the user's hardware ([[User Hardware Specifications]])
- Good system management, no bloating
- Fine-tuned
- TUI greeter
- Optimized Hyprland ecosystem
- Capable of running:
    - LLMs
    - Steam games
    - Coding (Neovim - nvchad or lunar vim)
- Easy to install and bundle everything needed for a fresh install.
- The purpose is to use it on the main laptop to plan, organize, code, and develop the Cognitive Internet Project.

## Development Areas

- Kernel Development
- AI/LLM Development
- Web Development
- Hardware
- Note Taking (Obsidian.md or alternatives)
